2. JavaScript Class Methods

Method Parameter Deskripsi Contoh
next() - Slide berikutnya carousel.next()
prev() - Slide sebelumnya carousel.prev()
goToSlide() index: number Pindah ke slide tertentu carousel.goToSlide(2)
updateCarousel() - Update posisi carousel carousel.updateCarousel()
autoPlay() - Mulai auto-play carousel.autoPlay()

Class Name Fungsi Properties

5. Fitur Advanced (Optional)

A. Keyboard Navigation

// Tambahkan di student-affairs.js:
document.addEventListener('keydown', (e) => {
  if (e.key === 'ArrowLeft') carousel.prev();
  if (e.key === 'ArrowRight') carousel.next();
});

B. Touch/Swipe Support

// Untuk mobile swipe (menggunakan Hammer.js atau custom):
let startX = 0;
carousel.addEventListener('touchstart', (e) => {
  startX = e.touches[0].clientX;
});
carousel.addEventListener('touchend', (e) => {
  let endX = e.changedTouches[0].clientX;
  if (startX > endX) carousel.next();
  if (startX < endX) carousel.prev();
});

C. Pause Auto-Play on Hover

// Tambahkan di student-affairs.js:
carousel.addEventListener('mouseenter', () => {
  // Pause auto-play saat hover
});

carousel.addEventListener('mouseleave', () => {
  // Resume auto-play saat mouse out
});

7. Best Practices

  • Gunakan semantic HTML5 elements
  • Mobile-first responsive design
  • Optimize images sebelum upload
  • Test di berbagai browsers
  • Gunakan CSS Grid/Flexbox untuk layout
  • Avoid inline styles, gunakan CSS classes
  • Minify CSS & JS untuk production
  • Implement lazy loading untuk gambar
  • Ensure accessibility (ARIA labels)
  • Keep code DRY (Don't Repeat Yourself)

8. Troubleshooting Checklist

Problem Check Solution
Carousel tidak jalan Script ter-link? Cek <script src="..."> tag
Gambar tidak muncul Path benar? Verifikasi ../Kumpulan foto/nama.jpg
Styling berantakan CSS ter-link? Clear cache Ctrl+Shift+Del
Buttons tidak clickable z-index layering? Tambah z-index di CSS
Mobile responsive error Meta viewport tag? Pastikan ada viewport tag
