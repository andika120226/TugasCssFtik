EDIT KONTEN:
  1. Buka StudentAffairs.html dengan text editor
  2. Cari teks yang ingin diubah
  3. Edit sesuai kebutuhan
  4. Save file (Ctrl+S)
  5. Refresh browser (F5)

GANTI GAMBAR:
  1. Ubah path di <img src="...">
  2. Gunakan path: ../Kumpulan foto/nama-gambar.jpg
  3. Pastikan file gambar ada di folder

UBAH WARNA:
  1. Buka student-affairs-styles.css
  2. Cari warna hex (#c41e3a, #8b0000, dll)
  3. Ganti dengan warna baru
  4. Refresh browser

TAMBAH SLIDE:
  1. Copy <div class="carousel-item"> block
  2. Paste setelah slide terakhir
  3. Ubah konten (gambar, teks, dll)
  4. Tambah <span class="dot"></span> di indicators
  5. Update counter (1/5 ‚Üí 1/6)

Lihat PANDUAN_KUSTOMISASI.md untuk detail lebih lanjut!

                        üìû TROUBLESHOOTING
================================================================================

Masalah: Carousel tidak jalan
Solusi:
  - Buka DevTools (F12)
  - Cek tab Console untuk error messages
  - Pastikan student-affairs.js ter-link dengan benar

Masalah: Gambar tidak tampil
Solusi:
  - Verifikasi path: ../Kumpulan foto/nama.jpg
  - Pastikan file gambar ada di folder
  - Check console untuk 404 errors

Masalah: Styling berantakan
Solusi:
  - Clear browser cache (Ctrl+Shift+Delete)
  - Hard refresh page (Ctrl+Shift+R)
  - Pastikan CSS file ter-link dengan benar

Masalah: Mobile tidak responsive
Solusi:
  - Pastikan viewport meta tag ada di HTML
  - Test dengan DevTools Responsive Mode (F12 ‚Üí Ctrl+Shift+M)
  - Check CSS media queries

Lihat DOKUMENTASI_STUDENT_AFFAIRS.md untuk troubleshooting lebih lengkap!
